## 🔬 Overview  
A novel Reservoir Computing architecture that:  
- Implements **multi-scale chaotic attractor partitioning** via hierarchical k-means  
- Combines **fractal connectivity** with **neuroscience principles** (small-world wiring, excitation-inhibition balance)  
- Outperforms standard ESNs on long-horizon forecasting tasks
